# Hub3

Hub3 is a Solana-first platform for publishing public GitHub repositories into permanent storage, registering provenance on-chain, and browsing published repository state through human and agent-friendly APIs.

## Workspace

- `apps/web` - Next.js frontend
- `apps/api` - Fastify backend
- `packages/shared` - shared schemas and contracts
- `programs/repo_registry` - Anchor Solana program
- `clients/ts/repo_registry` - checked-in generated client stub
